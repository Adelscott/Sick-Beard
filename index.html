    Version de Sickbeard pour les français

    Project maintained by sarakha63 Hosted on GitHub Pages — Theme by mattgraham

    Je fais ce sujet pour présenter une version de sickbeard qui va plaire à la communauté. J'en ai déjà parlé sur de nombreux sujet mais il est temps de faire une présentation offcicielle.

    Fonctionnalités supplémentaires de cette version :

    -Cette version permet pour chaque série de choisir si on la veut en VO ou en VF.

    -La plupart des providers permettent de télécharger en VF

    -Binnews à été rajouté

    -T411 a été rajouté

    -Cpasbien a été rajouté

    -GKS a été rajouté

    -L'envoi automatique de torrent a transmission, utorrent, déluge et DS Download a été implémenté en plus du black hole

    -Subliminal a été implémenté pour télécharger les sous titre.

    -De nombreuses modifications d'interfaces

    -Chaque épisode téléchargé et renseigné automatiquement avec la langue téléchargée pour savoir ou en en est.

    -Pour chaque épisode possédé on peut sélectionner sa langue au cas ou elle a pas été renseigné automatiquement ou si l'épisode a été téléchargé manuellement. Lors d'une première install, sickbeard ne peut pas deviner en quel langue sont les épisodes, il faut donc penser a le faire pour les épisodes déjà possédées (on peut selectionner plusieurs épisodes pour le faire)

    -Logo pour les chaines

    -Gestion de trakt et import auto des Watchlist (avec possibilité de dire skip, prendre touts les épisodes, seulement le premier ou les 3 premiers)

    -Rajout the pirate bay (toujours en beta)

    -Rajout de Kick ass torrent (beta)

    -Rajout de la possibilité d'avoir les sous titre sans language code

    -Rajout de la notion de preferred torrent preferred newsgroup

    -Rajout de la gestion des failed (une recherche relancé ne prendra pas les liens déjà téléchargés et non post processés de cet épsiode (on peut remettre la liste des liens à jour))

    -Nettoyage récursif des repertoires dans le cas ou on utilise pas le move associated files

    -Rajout d'un repertoire a part pour les torrents pour le post processing

    Installation :

    Rien de plus simple, il faut installer le spk git fourni par synocommunity puis utiliser le spk sickbeard custom distribué par synocommunity et utiliser ces infos à l'installation :

    adresse git : git://github.com/sarakha63/Sick-Beard.git

    branche : master

    De nombreuses personnes, tombent sur ce sujet en cherchant sur google, donc je précise que l'installation décrite ci dessus est pour les nas synology, il n'existe pas de spk custom pour windows. La solution pour windows est d'installer python et cheetah et de télécharger les sources. Une version compilée arrivera quand elle sera dispo je mettrais le lien sur le sujet.
